以下为乐队成员的字典。**键**为第一部分（如“john”），其关联的**值**为第二部分（如“rhythm guitar”）。

```python
band = {
  'john' : 'rhythm guitar',
  'paul' : 'bass guitar',
  'george' : 'lead guitar',
  'ringo' : 'bass guitar'
	}
```

下面显示了如何向字典添加键值对：

```python
# Add a key:value pair
band['yoko'] = 'vocals'
```

下面显示了如何从字典中移除键值对：

```python
# Remove a key:value pair
del band['paul']
```
